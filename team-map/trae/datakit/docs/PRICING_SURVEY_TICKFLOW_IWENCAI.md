### 1.3 🔴 定价盲区（核心发现）

**注册版的具体价格未在任何公开渠道找到。**

调查过程：
- `tickflow.org/pricing/` → JS 动态渲染页面，WebFetch 无法抓取实际内容
- `docs.tickflow.org` → 文档站不含定价页，仅技术文档
- GitHub README → 只描述功能，不写价格
- 知乎作者原文（zhuanlan.zhihu.com/p/2015793350209450091）→ 仅分"免费/高级"两类，**未列价格**
- V2EX 推广帖 → 同知乎，仅描述功能差异，**未列价格**
- 火山引擎开发者文章 → 引用 TickFlow 作为 AKShare 收费版的例子，称"价格合理"，但**无数字**
- CSDN 2026年文章 → 称"TickFlow（akshare收费版），因其数据稳定、价格合理"，**无数字**

**可确认的事实：**
1. ✅ 注册版是**付费产品**（AKShare 作者的商业化项目）
2. ✅ 价格被跨文章描述为"合理"
3. ❌ 具体月费/年费数字 — **全网未公开**
4. ❌ 是否存在多级套餐（个人/专业/企业）— **未确认**

### 1.4 WebSocket 权限分级

从文档 LLMs.txt 中发现的权限分层线索：

| WebSocket 版本 | 接口 | 所需权限 |
|----------------|------|---------|
| 旧版 `Quotes` | 仅推送行情（无深度） | 基础实时行情权限 |
| 新版 `Stream` → `quotes` 频道 | 实时行情推送 | **WebSocket 实时行情权限** |
| 新版 `Stream` → `depth` 频道 | 市场深度/盘口 | **市场深度权限** |

暗示注册版内部还有细分权限层，可能对应不同价格档位。

## 二、i问财 SkillHub 定价调查

### 2.1 产品家族识别

同花顺旗下存在 **三个不同的数据产品**，容易混淆：

```
同花顺数据产品矩阵
├── ① 同花顺量化接口 (Quant API)
│   └── quantapi.10jqka.com.cn
│   └── 传统 REST API，面向量化策略开发者
│   └── 有明确免费版 + 付费版
│
├── ② iFinD 金融终端
│   └── 机构级产品，年费数万~数十万元
│   └── 不在本项目预算范围
│
└── ③ i问财 SkillHub
    └── iwencai.com/skillhub
    └── AI Agent 技能市场，面向 OpenClaw 等 Agent 平台
    └── OpenAPI Key 格式：sk-proj-...
    └── 定价信息 —— ❌ 未公开
```

### 2.2 版本 ①：同花顺量化接口 (Quant API)

#### 免费版额度（已确认，来源：quantapi.10jqka.com.cn 权限说明）

| 数据类别 | 具体数据类型 | 免费月额度 |
|----------|-------------|-----------|
| 行情数据 | 实时行情 | **300 万次/月** |
| 行情数据 | 日内快照 | **200 万次/月** |
| 行情数据 | 历史行情 | **100 万次/月** |
| 行情数据 | 高频序列 | **150 万次/月** |
| 基本面数据 | 日期序列 | **60 万次/月** |
| 基本面数据 | 基础数据 | **60 万次/月** |
| 基本面数据 | 专题报表 | **60 万次/月** |

额外限制（来源：CSDN 文章）：
- IP 频率限制：同一 IP 每天超过 **1 万次调用** 会触发限制
- 实时行情仅 **5 秒级快照**
- 日K 历史数据最多 **3 年**
- 不支持盘口数据
- 可申请临时扩容（约 +50%）

#### 付费版

| 套餐 | 说明 |
|------|------|
| 基础版 | 具体价格被 CSDN 付费墙屏蔽，月调用量提升，数据权限扩展 |
| 专业版 | 同上，更高调用量 + 更多数据维度 |
| 企业定制版 | 按需定制 |

> ⚠️ **付费版具体价格未公开获取。** CSDN 文章 `lake5/article/details/155545555` 的定价部分被付费墙屏蔽。

#### 🔴 定价盲区

调查过程：

### 4.2 风险提示

1. **TickFlow** 是个人开发者作品，定价策略可能随时调整。如果当前月费很便宜（推测 ¥30-80/月），建议提前确认是否有涨价计划。
2. **SkillHub** 如果目前免费，可能在正式商业化后转为收费。建议在接入代码中做好降级准备（如 Fallback 到免费 iwencai.com Web 查询）。
3. 两个 API Key 都明文存在于 `.env` 中，**存在泄露风险**。建议迁移到 `~/.openclaw/keys.env` 外挂引用。
