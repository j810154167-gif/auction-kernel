# 博弈意识 — A 股竞价中的资金心理博弈

## 起源 (0618)

A股 T+1 无卖空 + 中小资本无对冲工具 → 唯一防御 = 不下注在错的标的上。
不是预测方向，是检测**真金白银的多空对抗**。

## 三闸对照逻辑（0618 原设计）

每个 Gate 都有自己的参照物，不是系数：

```
① 价 对照 昨日VWAP区间   → 今天的价格和昨天的真实成交比
② 量 对照 昨日竞价量     → 今天的参与度和昨天比
③ 梯队 对照 板块涨幅      → 个股和同题材的其他票比
```

三维参照物告诉你"竞价行为是否可解释"。任何一维解释不了=在作假。
chg×3 把对照逻辑毁了——把不同参照物的信号压缩成一个数进行跨票比较。

## 量化资金的日内博弈

### 做市商/游资的心理底线

低开幅度可控——一旦跌破做市商控制区间，触发量化踩踏，这是各方不愿看到的。
他们的心理是给散户一个自我欺骗的信号：低开但没破位，"可能是洗盘"。

### L0 防守区间

昨日 [最低价, 爆量均价] = 做市商/控制者愿意承受的底线：

- 低开但在区间内 = **故意低开骗筹** = 机会，不是风险
- 跌破区间下沿 = 控制失效 = 踩踏开始

D1 中接入 L0 区间标记（🟢在区间上 / ■区间内 / 🔽区间下），一列一个字符。

### 参数化是量化的靶子

设 top 7% 扣分 → 量化拉到 6.8% 避开。设 bottom 3% 扣分 → 量化砸到 2.9%。
任何静态参数都是量化套利的标的。人不被算法牵着走，算法就没法套人。
