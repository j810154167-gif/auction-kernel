# 涨停前回溯分析 — 无效结论

## 实验: 0714 Top10 涨停前10日日K分析

尝试通过涨停前多日K线检测"机械量化对倒"和"僵尸交易"信号，为人类剃刀提供额外标记。

### 测试信号（全部尝试后否定）

| 信号 | 检测方法 | 结果 |
|------|------|------|
| 僵尸量 | 前5日均量 < 50k | 4/10触发, 区分度低 |
| 机械量CV | 前5日量 CV < 0.3 | 8/10触发, 太宽无区分 |
| 僵直振幅 | 前5日振幅 < 2% | 0/10触发, 无效 |
| 爆发比 | 涨停量/前5日均量 > 3x | 4/10触发 |
| 博弈比 | 前5日最大/均量 < 1.5 | 0/10触发 |
| 振幅>3%天 | 前10天有几天振幅>3% | 低价股失真 |
| 量价背离 | 前10天有无量增30%+价反向 | 9/10=0天 |

### 结论

**A股涨停标的在涨停前确实都长一个样——安静、机械、无人关注。涨停本身就是第一根博弈信号。** 前日日K回溯不能区分标的，因为所有涨停标的涨停前都没有博弈痕迹。

放弃此优化方向。不纳入 Scoring 引擎。
