# 强化防守模块

悬置状态按需深度分析。引擎: `core/defense/reinforce.py`。点火词: `强化 SYM [成本]`。

## 触发场景

- 收盘 < VWAP 但 L2 未触发 → 悬置（高位量>低位量）
- 博弈带买盘占比衰减 → 机构买入意愿消退
- 外部催化剂等待 → 题材发酵
- "感觉不对但规则说持有"

## 引擎命令

```bash
cd profiles-defense
python3 scripts/defense_engine.py 强化 001229.SZ 44.07
```

产出: `handoff/{date}/reinforce_{SYM}_{HHMM}.json`

## 全周期量能分析

### 量能地图
| 区域 | 特征 | 防守含义 |
|------|------|------|
| 博弈带 | 最近2-3日大量换手 | 当前战场 |
| 入场换手区 | 入场日量能集中 | 成本附近支撑 |
| ⚠ 空区 | 一字板/缩量跳过 | 跌破→自由落体 |
| 蓄力区 | 起涨前大量横盘 | 强支撑 |

### 大单方向分析（价格带买/卖占比）
| 信号 | 判定 |
|------|------|
| 低价带买盘>60% + 占全天量>30% | 机构吸筹 |
| 深水区100%卖 + 随后低价带买盘涌入 | V底吸筹 ← 蜜罐不成立 |
| 高价带（成本以上）卖盘>买盘 | 积累盘获利了结，非出货 |

大单定义: >3000手 或 >前20分钟均量×3。

### 悬置判定
| 条件 | 含义 |
|------|------|
| 收盘 < VWAP | 高位出货（量在收盘价上方堆积） |
| 博弈带买盘 < 55% | 买方主导消退 |
| 机构净买递减 | 大单从+15k→+5k |

## 悬置→台账流程

1. `强化 001229.SZ 44.07` → 深度分析 → 输出 reinforce_*.json
2. 你阅读报告 → 决定是否标记悬置
3. 改 `config/ledger.csv` 一行: `active→suspended`，更新 `entry_price=今收`
4. `建区间` 重跑 → 下次 `监控` 自动标记 `⚠悬置`

**悬置只能由人类标记。机器报事实，不替人决策。**

## 关键案例: 魅视科技 0709-0713

### 0709 开板日
- 深水区[40-42): 8k手 100%卖 → 恐慌抛售，散户被震出
- 积累区[42-44): 79k手 66%买 → 机构重仓吸筹(占全天55%)
- 成本带[44-45): 37k手 57%卖 → 积累盘获利了结
- 结论: V底吸筹，蜜罐不成立。机构在42-44真金白银进场，成本≈43

### 0710 博弈日
- 大单净+15568 → 全力拉，价格上移

### 0713 今日
- 大单净+5333 → 衰减到1/3
- 博弈带买盘65%→57% → 意愿消退
- 下午首现尾盘砸: 1888手@15:00
- 收盘<VWAP → 首日高位量+低位收
- 悬置: 机构在等朱雀火箭发酵→高开派发。信号未到前横盘减仓

### 0714 结果
- 朱雀火箭未发酵 → 开盘即跌穿，持续19分钟破位 → 🔴卖出
- 全天跌-7%，成本44.07→43.x 浮亏
